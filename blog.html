Site Launch!

March 25, 2014

TEST-------- Hey everybody! Welcome to the first website I have built by writing the code for it. This site isn't complete yet, nevertheless, thank you for visiting. If you're reading this on March 25, 2014, then you are among the very first visitors to this site. I plan to add a few more things and features over the next few days, including the "About" and "Projects" page up and running.

I have several purposes for this site. I hope to use this website as a general hub for things I do on the web. I used to design texture hacks for Super Smash Bros. Brawl over at Brawl Vault, and I have recently began to write CSS Styles over at Userstyles.org. Anyway, I want to use this site to connect that type of stuff at one central location. As a matter of fact, this site itself is partly practice for me and a place for me to sharpen my web development abilities. I plan for this to be an ongoing project, so you can probably expect to see some changes around here soon enough.
